# P2_infiniTIC

## Descripción

Proyecto que implementa una arquitectura multi-entorno (dev, staging, prod)
usando Terraform con backend remoto en Azure Blob Storage.

## Características

- Terraform modular
- Backend remoto
- Separación por entornos
- Pipeline CI/CD en Azure DevOps

## Nota

Se utilizaron recursos gratuitos (F1) para evitar costos,
manteniendo la arquitectura y buenas prácticas.
